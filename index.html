Welcome

Welcome to our static web application hosted on AWS S3. This site demonstrates a clean, accessible, and responsive design built with plain HTML, CSS, and JavaScript.

About

This application is a self-contained static website designed for deployment on AWS S3 static website hosting. It requires no server-side processing and runs entirely in the browser.

Features

  • Responsive design that works on desktop and mobile
  • Smooth scroll navigation between sections
  • Accessible markup following WCAG 2.1 AA standards
  • Custom error page for a polished user experience
  • Lightweight and fast loading with no external dependencies
